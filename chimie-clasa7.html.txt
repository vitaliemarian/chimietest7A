        <h2>NORMELE DE LUCRU ÎN CABINETUL (LABORATORUL) DE CHIMIE</h2>
        <div class="vas-laborator">
            <h3>Reguli esențiale de securitate în laborator</h3>
            <div class="regula-securitate">1. Înainte de a începe lucrul, îmbracă halatul, la indicația profesorului, poți folosi și alte mijloace de protecție (ochelari, mănuși speciale etc.).</div>
            <div class="regula-securitate">2. Citește cu atenție eticheta vasului din care urmează să iei reactivul necesar.</div>
            <div class="regula-securitate">3. Nu lăsa deschise recipientele cu reactivi. Nu schimba cu locul dopurile și pipetele pentru reactivi.</div>
            <div class="regula-securitate">4. Ia doar cantitatea de reactiv indicată în instrucțiune. De obicei, pentru lucrarea cu cantități mici de substanță (până la 1 g pentru substanțele solide sau 1 ml pentru cele lichide).</div>
            <div class="regula-securitate">5. Resturile de reactivi nu le pune înapoi în vasul din care le-ai luat, ci toarnă-le într-un balon special pentru resturi sau înapoiază-le laborantului.</div>
            <div class="regula-securitate">6. Nu lua reactivii cu mâna, ci cu instrumentele speciale (linguriță, spatulă).</div>
            <div class="regula-securitate">7. Se interzice categoric ca reactivii să fie gustați, deoarece majoritatea lor sunt otrăvitori.</div>
            <div class="regula-securitate">8. Miroase substanțele se va face în felul următor: prin mișcări ușoare ale mâinii deasupra vasului, vaporii care se degajă vor fi îndreptați spre nas (fig. 1.2, pag. 23).</div>
            <div class="regula-securitate">9. Pentru a încălzi la flacără eprubeta cu substanță, prinde eprubeta în clema pentru eprubete (numită impropriu cleștar), încălzește-o slab pe toată suprafața, apoi în partea necesară.</div>
            <div class="regula-securitate">10. În timpul încălzirii, ține eprubeta ușor înclinată și nu îndrepta gura eprubetei spre tine, uneori, în timpul fierberii, sar stropi de lichid.</div>
            <div class="regula-securitate">11. Dacă întâmplător s-a vărsat reactiv pe piele sau pe haine, anunță imediat profesorul (sau laborantul).</div>
            <div class="regula-securitate">12. Obiectele fierbinți nu le pune pe masă, ci pe suport special.</div>
            <div class="regula-securitate">13. Nu efectua nicio experiență în afara lucrării date.</div>
            <p><strong>După terminarea experiențelor,</strong> vei face curaț la locul de lucru. Vasele le vei spăla cu apă, folosind, pentru curățare, perii speciale.</p>
            <p><strong>Elevii sunt obligați să respecte aceste reguli, nerespectarea lor poate avea urmări grave: arsuri, răniri, intoxicații.</strong></p>
        </div>

        <h2>CAPITOLUL 1. CHIMIA – ȘTIINȚA DESPRE SUBSTANȚE</h2>
        <p><strong>După studierea acestui capitol, vei fi capabil:</strong></p>
        <ul>
            <li>să definești noțiunile de atom, moleculă, substanță, element chimic, simbol chimic, formulă chimică, masă atomică și masă moleculară relativă;</li>
            <li>să diferențiezi corpuri și substanțe, atomi și molecule, substanțe simple și compuse, metale și nemetale, substanțe organice și anorganice;</li>
            <li>să utilizezi sistemul periodic pentru identificarea informației despre elementele chimice;</li>
            <li>să descrii obiectul de studiu al chimiei, să cunoști regulile de tehnică a securității în laboratorul de chimie și utilajul de laborator, să afli proprietățile fizice și utilizarea unor substanțe, să definești modul în care acționează legea constanței compoziției;</li>
            <li>să exemplifici cum contribuie chimia la creșterea calității vieții;</li>
        </ul>

        <h3>1.1 Substanța – formă de existență a materiei</h3>
        <p>Din cele studiate până acum la fizică, biologie și geografie, am aflat că tot ce ne înconjoară este format din <strong>materie</strong>.</p>
        <div class="notiune">
            <p><strong>Materia reprezintă realitatea care există independent de noi și constituie izvorul tuturor transformărilor din natură.</strong></p>
        </div>
        <p><strong>Corpuri și substanțe.</strong> Întreaga lume care ne înconjoară este alcătuită dintr-o multitudine de obiecte sau corpuri fizice.</p>
        <div class="notiune">
            <p><strong>Corp fizic</strong> este orice obiect ce are <strong>masă, volum și alte caracteristici</strong>.</p>
        </div>
        <p>Astfel, cartea, pixul, lingura, paharul, calculatorul, automobilul s.a. sunt corpuri fizice (fig. 1.1).</p>
        <div class="fig">
            <p><em>Fig. 1.1 Diferite corpuri fizice</em></p>
        </div>
        <p>Prin ce se deosebesc toate aceste obiecte?</p>
        <p>Corpurile date au forme, dimensiuni și culori diferite.</p>

        <h3>Contribuția chimiei la sporirea calității vieții</h3>
        <p>Chimia contribuie la creșterea calității vieții prin producerea medicamentelor, vitaminelor, materialelor plastice, îngrășămintelor chimice, combustibililor, materialelor de construcție etc.</p>
        <p>Totuși, utilizarea incorectă a substanțelor chimice poate duce la poluare și efecte toxice asupra mediului și sănătății.</p>

        <h2>LUCRAREA PRACTICĂ NR. 1. VASE ȘI INSTRUMENTE DE LABORATOR</h2>
        <div class="vas-laborator">
            <h3>Vase și instrumente esențiale</h3>
            <ul>
                <li><strong>Eprubetă</strong> – pentru reacții în cantități mici, încălzire</li>
                <li><strong>Pahar de precipitații</strong> – pentru măsurarea volumului de lichide</li>
                <li><strong>Balon Erlenmeyer</strong> – pentru reacții, titrări</li>
                <li><strong>Cilindru gradat</strong> – pentru măsurarea precisă a volumului</li>
                <li><strong>Pâlnie</strong> – pentru filtrare, turnare</li>
                <li><strong>Clește pentru eprubete</strong> – pentru ținerea eprubetei la încălzire</li>
                <li><strong>Spirtieră</strong> – sursă de căldură</li>
            </ul>
            <p><strong>Regulă importantă:</strong> După terminarea experienței, <strong>vasele se spală cu apă, folosind perii speciale</strong>.</p>
        </div>

        <h3>1.4 Proprietățile substanțelor</h3>
        <ul>
            <li><strong>Proprietăți fizice:</strong> culoare, miros, solubilitate, densitate, conductibilitate termică și electrică</li>
            <li><strong>Proprietăți chimice:</strong> capacitatea de a reacționa cu alte substanțe</li>
            <li><strong>Proprietăți fiziologice:</strong> acțiunea asupra organismului (toxicitate, inflamabilitate)</li>
        </ul>

        <h3>1.5 Metodele de cunoaștere în chimie</h3>
        <p>Observare → Informare → Descriere → Măsurare → Meditație → Experiment</p>

        <h3>1.6 Fenomene fizice și chimice</h3>
        <ul>
            <li><strong>Fenomene fizice:</strong> nu schimbă compoziția substanței (topire, evaporare, spargere)</li>
            <li><strong>Fenomene chimice:</strong> schimbă compoziția (ardere, înnegrire, ruginire)</li>
        </ul>
        <p><strong>Semne ale reacțiilor chimice:</strong> degajare de gaz, formare de precipitat, schimbare de culoare, degajare/absorbție de căldură</p>

        <h3>1.7 Atomul. Molecula</h3>
        <ul>
            <li><strong>Atom</strong> = cea mai mică particulă indivizibilă a substanței</li>
            <li><strong>Moleculă</strong> = grup de atomi legați chimic</li>
            <li><strong>Substanțe simple</strong> = formate dintr-un singur tip de atomi (ex: O₂, Fe)</li>
            <li><strong>Substanțe compuse</strong> = formate din mai multe tipuri de atomi (ex: H₂O, CO₂)</li>
        </ul>

        <h3>1.8 Elementul chimic</h3>
        <p><strong>Element chimic</strong> = totalitatea atomilor cu aceleași proprietăți chimice și fizice.</p>
        <p>Astăzi sunt cunoscute <strong>118 elemente chimice</strong>.</p>

        <h3>1.9 Simbolurile chimice</h3>
        <p>Fiecare element are un <strong>simbol chimic</strong>: 1–2 litere, prima majusculă (ex: H, O, Fe, Au).</p>
        <p>Simbolul reprezintă atât elementul, cât și <strong>un atom al acestuia</strong>.</p>

        <h3>1.10 Masa atomică relativă</h3>
        <p>Se raportează la <strong>1/12 din masa unui atom de carbon-12</strong>.</p>

        <h3>1.11 Sistemul periodic al elementelor</h3>
        <ul>
            <li>7 <strong>perioade</strong> (rânduri orizontale)</li>
            <li>18 <strong>grupe</strong> (coloane verticale)</li>
            <li><strong>Metale</strong> (88%) – în partea stângă</li>
            <li><strong>Nemetale</strong> – în partea dreaptă</li>
        </ul>

        <h3>1.12 Metale și nemetale</h3>
        <ul>
            <li><strong>Metale:</strong> solide, lucioase, bune conductoare de căldură și electricitate, maleabile</li>
            <li><strong>Nemetale:</strong> gaze, lichide sau solide fragile, proaste conductoare</li>
        </ul>

        <h3>1.13 Formula chimică. Masa moleculară relativă</h3>
        <p><strong>Formula chimică</strong> arată compoziția moleculei (ex: H₂O = 2 atomi H + 1 atom O).</p>
        <p><strong>Masa moleculară relativă</strong> = suma maselor atomice relative ale atomilor din moleculă.</p>
        <p><strong>Legea constanței compoziției:</strong> orice substanță chimică are o compoziție constantă.</p>

    function verificaTest() {
        let scor = 0;
        const greseli = [];

        intrebari.forEach((q, i) => {
            let raspuns = null;
            let corect = false;

            if (q.tip === 'radio') {
                const sel = document.querySelector(`input[name="q${i}"]:checked`);
                raspuns = sel ? sel.value : '—';
                corect = raspuns === q.corect;
            } else if (q.tip === 'text') {
                const input = document.querySelector(`input[name="q${i}"]`);
                const val = input ? input.value.trim().toLowerCase() : '';
                raspuns = val || '—';
                corect = q.corect.some(c => c.toLowerCase() === val);
            } else if (q.tip === 'multi') {
                const checked = Array.from(document.querySelectorAll(`input[name="q${i}"]:checked`)).map(c => c.value);
                raspuns = checked.length ? checked.sort().join(', ') : '—';
                const corectSortat = q.corect.sort().join(', ');
                corect = raspuns === corectSortat;
            }

            if (corect) {
                scor++;
            } else {
                greseli.push({
                    intrebare: q.intrebare,
                    raspuns: raspuns,
                    corect: Array.isArray(q.corect) ? q.corect.join(', ') : q.corect,
                    explicatie: q.explicatie
                });
            }
        });

        document.getElementById('pagina-test').style.display = 'none';
        document.getElementById('pagina-rezultate').style.display = 'block';
        document.getElementById('btnTrimite').style.display = 'none';
        document.getElementById('btnInapoi').style.display = 'block';

        document.getElementById('scorFinal').textContent = `${scor}/30`;

        const procent = scor / 30;
        let nota = 1;
        if (procent >= 0.95) nota = 10;
        else if (procent >= 0.90) nota = 9;
        else if (procent >= 0.80) nota = 8;
        else if (procent >= 0.70) nota = 7;
        else if (procent >= 0.60) nota = 6;
        else if (procent >= 0.50) nota = 5;
        else if (procent >= 0.40) nota = 4;
        else if (procent >= 0.30) nota = 3;
        else if (procent >= 0.20) nota = 2;
        else nota = 1;

        document.getElementById('notaFinala').textContent = nota;
        document.getElementById('notaFinala').style.color = nota >= 6 ? '#32CD32' : '#e74c3c';

        if (greseli.length > 0) {
            document.getElementById('listaGreseli').style.display = 'block';
            const html = greseli.map(g => `
                <div class="gresala">
                    <strong>Întrebare:</strong> ${g.intrebare}<br>
                    <strong>Răspunsul tău:</strong> <span style="color:#e74c3c;">${g.raspuns}</span><br>
                    <strong>Răspuns corect:</strong> <span style="color:#32CD32;"><strong>${g.corect}</strong></span>
                    <div class="explicatie"><strong>De ce e corect?</strong> ${g.explicatie}</div>
                </div>
            `).join('');
            document.getElementById('listaGreseli').innerHTML = '<h3>Întrebări greșite:</h3>' + html;
        }
    }
